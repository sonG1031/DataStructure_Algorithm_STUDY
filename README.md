### 코딩을 잘하고 싶다. 어떻게 해야 할까?

코딩을 많이 해야 한다.

어떤 알고리즘의 원리와 증명을 이해하는 것은 중요하지만 **응용하는 것이 더 중요**하다.

-   인생은 짧고, 시간은 없고, 할 일은 많다.
-   도움을 받는 것과 이해하는 것, 검색을 하는 것도 매우 중요한 능력.
-   고민의 시간은
    -   처음 공부할 때는 2-3일
